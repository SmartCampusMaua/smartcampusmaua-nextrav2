# TimeSeries API


## Introdução

A API SmartCampus Mauá permite acessar dados de dispositivos conectados ao sistema, fornecendo informações detalhadas sobre suas medições, como luminosidade, temperatura, umidade, movimento e outros. A API se baseia em dados de séries temporais, que podem ser consultados de acordo com o intervalo de tempo desejado.

Esta documentação oferece detalhes sobre como usar a API para consultar dados de dispositivos, com exemplos práticos e explicações claras.



---

## Endpoints

### 1. **Obter dados de dispositivos dos últimos 30 minutos**
Este endpoint permite consultar os dados de todos os dispositivos conectados nos últimos 30 minutos.

**URL: https://smartcampus-k8s.maua.br/api/timeseries/v0.3/IMT/LNS/{1}/{2}?interval=30**


\{1} - Tipo de Dispositivo:  
      - SmartLight  
      - WaterTankLevel  
      - GaugePressure  
      - Hydrometer  
      - EnergyMeter  
      - WeatherStation  

\{2} - DeviceId ou "all" para todos os dispositivos. 

### 2. **Obter dados de dispositivos em um intervalo de tempo específico**
Este endpoint permite consultar os dados de todos os dispositivos conectados em um determindao intervalo de tempo.

**URL: https://smartcampus-k8s.maua.br/api/timeseries/v0.3/IMT/LNS/{1}/{2}?interval={3}**

\{1} - Tipo de Dispositivo:  
      - SmartLight  
      - WaterTankLevel  
      - GaugePressure  
      - Hydrometer  
      - EnergyMeter  
      - WeatherStation  

\{2} - DeviceId ou "all" para todos os dispositivos.

\{3} - Tempo em minutos.
